Transcript Search Tips
Search Features

This search looks through all transcript content, including speaker dialogue, to help you find specific stories, topics, and quotes.

Advanced Search Options
  • Search specific fields: Use field names like speaker:John, title:integration, or words:school
  • Wildcards: Add * to match any characters, e.g., teach* finds "teacher," "teaching," etc.
  • Fuzzy match: Add ~1 after a word to find similar spellings, e.g., school~1
  • Boost terms: Add ^10 to make a term more important, e.g., desegregation^10 school
Search Examples
  • Mountain View Elementary - Find mentions of this school
  • teach* - Find teacher, teaching, teachers, etc.
  • speaker:McIntosh football - Find when McIntosh talked about football
  • desegregation experience - Find discussions of desegregation experiences

Note: Results show up to 3 matching segments per interview. Click "View in transcript" to see the full context.
